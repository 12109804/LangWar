# 🧩 Lang War - プログラミング言語スキル型パズルゲーム

## 📌 概要

**Lang War** は、プログラミング言語の「特性」や「思想」抽象概念をスキルとして落とし込んだ、テトリス型パズルゲームです。

プレイヤーはブロックを積み上げながら、各プログラミング言語に対応したスキルを駆使してライン消去やスコア獲得を目指します。

単なるアクションゲームではなく、**言語ごとの設計思想や特徴を直感的に理解できる**ことを目的としています。

---

## 🌱 開発背景

プログラミング学習を進める中で、理解を深めながら開発できる題材としてパズルゲームを選びました。
プログラミングは「組み合わせて構築する」「論理のズレがエラーになる」など、パズルと共通する抽象性があり、学習との相性が良いと考えたためです。
また、学習中に感じた 技術や言語の“キャラクター性” をゲームに取り入れ、async/await など実際の技術感覚を表現することで、独自性とリアリズムを重視しました。
パズルゲームはシンプルに見えて拡張性が高く、保守性・アップデート性を学べる点も魅力でした。
プレイヤーが触った瞬間に抽象概念を直感的に理解できるよう、学習者視点とサービス視点の両方を意識して設計しています。

---

## 🎮 ゲームの特徴

### 🧠 言語 = スキル

各プログラミング言語は、それぞれ独自のスキルを持ちゲーム内に登場します。

| 言語 | スキル例 | 特徴 |
| ------ | --------- | ------ |
| JavaScript | Async Await | 時間操作 |
| Python | Library Boost | ランダムスキル発動 |
| C | Pointer Access | ブロック指定 |
| Ruby | Duck Taiping | ??? |
| GO | Zero Compile | 開始時スキル発動可 |
| Java | JavaVM | ??? |

---

### ⚙️ スキルシステム

- スキルは一定条件で発動
- クールダウンあり
- プレイスタイルに影響

---

### 🧩 パズルシステム

- 基本はテトリス型(パターンは追加予定)
- ライン消去でスコア獲得
- 同時消去やコンボあり

---

## 🕹️ ゲームフロー

1. 言語（スキル）を選択
2. ブロックを操作して積む
3. 条件を満たしてスキル発動
4. ライン消去・コンボでスコアを稼ぐ
