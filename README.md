# 🧩 Lang War - プログラミング言語スキル型パズルゲーム

## 📌 概要
**Lang War** は、プログラミング言語の「特性」や「思想」抽象概念をスキルとして落とし込んだ、テトリス型パズルゲームです。

プレイヤーはブロックを積み上げながら、各プログラミング言語に対応したスキルを駆使してライン消去やスコア獲得を目指します。

単なるアクションゲームではなく、**言語ごとの設計思想や特徴を直感的に理解できる**ことを目的としています。

---

## 🎮 ゲームの特徴

### 🧠 言語 = スキル
各プログラミング言語は、それぞれ独自のスキルとしてゲーム内に登場します。

| 言語 | スキル例 | 特徴 |
|------|---------|------|
| JavaScript | 非同期処理 | 一定時間後にブロック消去 |
| Python | シンプル構文 | 操作が簡略化される |
| C | 低レベル操作 | ブロックを細かく分解できる |
| Rust | メモリ安全 | ミス時のペナルティ軽減 |
| Haskell | 関数型 | 自動連鎖が発生する |

---

### ⚙️ スキルシステム
- スキルは一定条件で発動
- クールダウンあり
- プレイスタイルに影響

---

### 🧩 パズルシステム
- 基本はテトリス型
- ライン消去でスコア獲得
- 同時消去やコンボあり

---

## 🕹️ ゲームフロー

1. 言語（スキル）を選択
2. ブロックを操作して積む
3. 条件を満たしてスキル発動
4. ライン消去・コンボでスコアを稼ぐ

---
